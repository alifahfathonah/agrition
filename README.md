# Agrition
Agrition is an e-commerce that sells agricultural products, livestock, agricultural needs, etc.

## Features
* **Buyer Features**
    * Add a cart
    * Create a store
    * Buy some stuff
    * Online payment
* **Seller Features**
    * Sell some items
    * Update stock
    * Receive purchases
    * Update the receipt number
* **Admin Features**
    * Organizing goods, sales, and shops
